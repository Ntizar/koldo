- **Cómo usarlo**:
  1. **Navegar por categoría**: Usa la tabla de contenidos del README para saltar a cualquier categoría.
  2. **Explorar APIs**: Cada categoría muestra una tabla con nombre, descripción y enlace a la API.
  3. **Ver documentación**: Cada enlace lleva a la página del actor en Apify con documentación completa, ejemplos y precios.
  4. **Ejecutar vía API**: Todos los actores se pueden ejecutar programáticamente a través de la API de Apify.
  5. **Programar ejecuciones**: Configura horarios automáticos para recopilación regular de datos.
  6. **Acceso directo**: Los enlaces incluyen tracking de afiliado (`?fpr=p2hrc6`) y apuntan a `apify.com`.
  7. **Categorías individuales**: Cada categoría tiene su propia página (ej. `./ai-apis-1208/`) con la lista completa de APIs de esa categoría.
- **Notas importantes**:
  - Todas las APIs están ordenadas alfabéticamente dentro de sus categorías.
  - Las descripciones están truncadas a ~150 caracteres; para detalles completos visita las páginas individuales.
  - El listado se genera automáticamente desde la Apify Store API.
