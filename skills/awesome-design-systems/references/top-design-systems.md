# Top Design Systems

Lista de los design systems más relevantes extraída de [alexpate/awesome-design-systems](https://github.com/alexpate/awesome-design-systems).

---

## 1. Stripe Design System

- **URL:** https://stripe.com/design
- **Components:** 👍 | **Voice & Tone:** 👍 | **Designers Kit:** 👍 | **Source code:** 🐙
- **Stack:** React, TypeScript
- **Descripción:** Considerado el gold standard de la industria. Componentes de pago y UI con documentación excepcional, animaciones fluidas y API visualmente clara.
- **Componentes destacados:** Buttons, forms, overlays, datatables, toasts, payment elements
- **Código fuente:** https://github.com/stripe/stripe-js / https://github.com/stripe/react-stripe-js

## 2. Linear

- **URL:** https://linear.app
- **Components:** 👍 | **Voice & Tone:** 👍 | **Source code:** 🐙
- **Stack:** React, TypeScript
- **Descripción:** Design system minimalista y rápido, enfocado en productividad de equipos de ingeniería. Attributed a su velocidad percibida y UI limpia.
- **Componentes destacados:** Keyboard shortcuts, command palette, issue tracking UI, timelines
- **Código fuente:** Parcialmente visible en su documentación de diseño

## 3. Vercel / Geist

- **URL:** https://vercel.com/geist
- **Components:** 👍 | **Voice & Tone:** 👍 | **Source code:** 🐙
- **Stack:** React, TypeScript
- **Descripción:** Sistema de diseño de Vercel que incluye la tipografía Geist (fuente open source). Minimalista, con énfasis en performance y developer experience.
- **Componentes destacados:** Typography (Geist Sans/Mono), icons, layout system, code blocks
- **Código fuente:** https://github.com/vercel/geist

## 4. Adobe Spectrum

- **URL:** https://spectrum.adobe.com
- **Components:** 👍 | **Voice & Tone:** 👍 | **Designers Kit:** 👍 | **Source code:** 🐙
- **Stack:** React, Vue, Angular, iOS, Android
- **Descripción:** Design system de Adobe con soporte multi-plataforma. Muy completo con kits de diseño para Figma y Sketch.
- **Componentes destacados:** Action bars, cards, dialogs, navigation, data tables, pickers
- **Código fuente:** https://github.com/adobe/react-spectrum

## 5. Atlassian Design System

- **URL:** https://atlassian.design
- **Components:** 👍 | **Voice & Tone:** 👍 | **Designers Kit:** 👍 | **Source code:** 🐙
- **Stack:** React, TypeScript
- **Descripción:** Sistema de Atlassian (Jira, Trello, Confluence). Enfocado en productos enterprise complejos con alta densidad de información.
- **Componentes destacados:** Tables, forms, navigation, feedback components, status indicators
- **Código fuente:** https://bitbucket.org/atlassian/atlassian-frontend-mirror/

## 6. AWS Cloudscape Design System

- **URL:** https://cloudscape.design
- **Components:** 👍 | **Voice & Tone:** 👍 | **Designers Kit:** 👍 | **Source code:** 🐙
- **Stack:** React, Vue, Angular
- **Descripción:** Design system de AWS para consolas y dashboards. Optimizado para aplicaciones enterprise con mucha data.
- **Componentes destacados:** Data tables, navigation, forms, status indicators, charts
- **Código fuente:** https://github.com/cloudscape-design/components

## 7. Microsoft Fluent UI

- **URL:** https://developer.microsoft.com/en-us/fluentui#/
- **Components:** 👍 | **Voice & Tone:** 👍 | **Designers Kit:** 👍 | **Source code:** 🐙
- **Stack:** React, Web Components, Angular, Office Add-ins
- **Descripción:** Design system de Microsoft con soporte multi-framework. Base de Office, Windows y Teams. Incluye guías de accesibilidad robustas.
- **Componentes destacados:** Breadcrumb, Button, CommandBar, Dialog, DetailsList, Tabs
- **Código fuente:** https://github.com/microsoft/fluentui

## 8. Chakra UI

- **URL:** https://chakra-ui.com
- **Components:** 👍 | **Voice & Tone:** — | **Source code:** 🐙
- **Stack:** React
- **Descripción:** Library modular de React con enfoque en accesibilidad por defecto y sistema de tokens personalizable. Muy popular por su simplicidad.
- **Componentes destacados:** Box, Flex, Grid, Button, Input, Modal, Toast, Spinner
- **Código fuente:** https://github.com/chakra-ui/chakra-ui

## 9. Ant Design

- **URL:** https://ant.design
- **Components:** 👍 | **Voice & Tone:** 👍 | **Designers Kit:** 👍 | **Source code:** 🐙
- **Stack:** React (también Angular, Vue, Flutter disponibles)
- **Descripción:** Design system de Alibaba con +60 componentes. Muy popular en China y en proyectos enterprise. Documentación en múltiples idiomas.
- **Componentes destacados:** Table, Form, Modal, DatePicker, Tree, Upload, Steps, Timeline
- **Código fuente:** https://github.com/ant-design/ant-design

## 10. Tailwind UI

- **URL:** https://tailwindui.com
- **Components:** 👍 | **Voice & Tone:** 👍 | **Source code:** 🐙
- **Stack:** React, Vue, HTML (basado en Tailwind CSS)
- **Descripción:** Colección de componentes creados por los autores de Tailwind CSS. No es un design system completo sino una librería de componentes premium basados en utilidades.
- **Componentes destacados:** Navigation, cards, forms, tables, modals, notifications
- **Código fuente:** Incluido con la compra (licencia premium)
