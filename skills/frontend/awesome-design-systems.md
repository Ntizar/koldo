# Awesome Design Systems

- **URL:** https://github.com/alexpate/awesome-design-systems
- **Categoría:** Frontend / Design Systems
- **¿Qué hace?:** La colección más completa de design systems del mundo (161+ sistemas documentados). Cada entrada incluye enlaces a documentación, componentes codificados, guías de voz y tono, y kits de diseño (Sketch/Figma). Etiquetado por: Components, Voice & Tone, Designers Kit, Source code.
- **Casos de uso:**
  - Referencia rápida para elegir un design system para un proyecto
  - Benchmark de cómo otras empresas estructuran sus sistemas de diseño
  - Inspiración para crear un design system propio
  - Aprender patrones de componentes y documentación
  - Encontrar sistemas open source para proyectos internos

- **Cómo integrarlo en proyectos:**
  1. Navegar la colección filtrando por tags (Components, Source code, etc.)
  2. Elegir un design system que coincida con tu stack (React, Vue, Angular, etc.)
  3. Clonar el repo del design system elegido para explorar componentes
  4. Adaptar tokens de diseño (colores, tipografía, spacing) a tu marca
  5. Seguir patrones de documentación del design system elegido
  6. Reutilizar patrones de componentes, no copiar código directamente
