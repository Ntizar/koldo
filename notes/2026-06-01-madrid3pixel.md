## Qué es
Visor isométrico interactivo de Madrid en estilo píxel art, inspirado en isometric.nyc.
Sistema de tiles procedurales con edificios 3D, ciclo día/noche y efectos atmosféricos.

## Stack
- HTML + Canvas 2D + Aurora CSS v5.1
- Vanilla JS (sin frameworks)
- GitHub Pages + Actions CI/CD
- 100% español (código, comentarios, UI)
